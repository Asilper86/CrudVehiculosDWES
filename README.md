# CrudVehiculosDWES

**Autor:** Adrián Siles Pérez  

## Descripción
**CrudVehiculosDWES** es un proyecto que permite gestionar vehículos mediante un CRUD. Además, cuenta con gestión de sesiones de usuarios, cada usuario debe iniciar sesión para poder acceder a sus vehículos y administrarlos.

Funcionalidades principales:  
- Registro y autenticación de usuarios.  
- Gestión de sesiones seguras.  
- Crear, leer, actualizar y eliminar vehículos.  
- Vehículos asociados a cada usuario.
